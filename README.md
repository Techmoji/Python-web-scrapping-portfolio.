# Web Scraping & Data Extraction Portfolio

This repository contains Python scripts demonstrating automated data extraction from both static and dynamic web pages. The projects focus on navigating HTML structures, handling JavaScript-rendered content, and processing raw text into structured, high-quality datasets.

## Projects Included

### 1. E-Commerce Data Extractor (Static Content)
* **File:** `scraper.py`
* **Tools:** Python, Requests, BeautifulSoup, Pandas
* **Description:** Extracts book titles, pricing, and stock availability from a hierarchical HTML structure. Cleans currency formatting and validates the data into a structured CSV format.

### 2. Dynamic Content Scraper (Infinite Scroll/AJAX)
* **File:** `selenium_scraper.py`
* **Tools:** Python, Selenium WebDriver, Pandas
* **Description:** Automates browser interactions to handle JavaScript-rendered content and infinite scroll pagination. Successfully extracts quote text and author details, outputting a cleaned CSV dataset.

## Core Competencies Demonstrated
* Python web scraping (BeautifulSoup, Selenium)
* Handling dynamic JS content (infinite scroll, AJAX)
* Data processing, cleaning, and normalization (Pandas)
* Delivering high-quality datasets in well-structured formats (CSV)
